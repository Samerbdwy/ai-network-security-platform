# 🛡️ AI Unified Network Security Platform

A unified security platform that combines network discovery scanning and AI-powered Cisco config hardening using Google Gemini AI.

---

## 🎯 What It Does

**Two tools in one platform:**

### Network Discovery Scanner
Enter any IP or domain → Scans for open ports and live devices on public targets. Detects services like SSH, HTTP, HTTPS, and DNS across different network configurations.

### Cisco Config Analyzer
Paste any Cisco running-config → AI instantly tells you:
- **SECURITY SCORE:** 0-100 rating
- **RISK LEVEL:** Critical / High / Medium / Low
- **VIOLATIONS:** Telnet, weak passwords, HTTP server, missing encryption, SNMP public
- **FIX:** Exact Cisco commands to harden the device

---

## 🛠️ Tech Stack

- Python + Flask (backend)
